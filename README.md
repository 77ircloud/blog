## 准备
- node.js环境
- git 环境
- clone项目

> windows 自行配置环境变量

## Hexo配置
全局安装hexo-cli
```
npm install hexo-cli -g
```
安装hexo
```
npm install hexo --save
```
安装依赖的包
```
npm install
```

## hexo命令

创建文章
```
$ hexo new "My New Post"
```

在本地查看
```
hexo s
```

生成静态文件
```
hexo g
```

发布到线上：
```
hexo d
```
